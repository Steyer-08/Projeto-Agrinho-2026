# 🌱 Guia Verde 

## 📌 Objetivo do projeto

O projeto **Guia Verde** foi desenvolvido para o tema do Agrinho:

**“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”**

O objetivo do site é mostrar como pequenas atitudes sustentáveis podem ajudar na produção de alimentos e na preservação do meio ambiente. O projeto apresenta informações sobre plantio em casa, reaproveitamento de resíduos orgânicos, reciclagem e descarte correto de materiais.

A proposta é incentivar práticas simples e conscientes, mostrando que é possível produzir alimentos de forma mais sustentável, reduzir desperdícios e reutilizar materiais no dia a dia.

## 🌾 Relação com o tema Agrinho

O tema Agrinho está presente no projeto ao relacionar o cultivo de alimentos com a sustentabilidade ambiental.

O site mostra que o agro e o meio ambiente podem caminhar juntos quando há cuidado com os recursos naturais, reaproveitamento de resíduos, redução do lixo e incentivo ao plantio consciente.

Por meio dos guias de plantio e reciclagem, o usuário aprende maneiras práticas de contribuir para um futuro mais sustentável.

## 🛠️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* HTML5
* CSS3
* JavaScript

## 🧩 Funcionalidades

O site possui:

* menu de navegação;
* layout responsivo para celular e computador;
* carrossel de alimentos;
* carrossel de materiais recicláveis;
* cards interativos com imagens;
* guias de plantio exibidos ao clicar nos alimentos;
* guias de descarte e reutilização exibidos ao clicar nos materiais;
* manipulação de conteúdo com JavaScript;
* rolagem suave entre as seções.

## ▶️ Instruções de uso

Para utilizar o projeto, siga os passos abaixo:

1. Baixe ou clone este repositório.
2. Abra a pasta do projeto no computador.
3. Abra o arquivo `index.html` em um navegador.
4. Navegue pelo site usando o menu.
5. Clique nos cards de alimentos para ver instruções de plantio.
6. Clique nos cards de materiais recicláveis para ver formas de descarte e reutilização.

Também é possível acessar o projeto pela página publicada no GitHub Pages.
